# make

## Overview
The make package simplifies the build process

## Features
- Allows you to build custom packages

## Installation
Install using KPM with:
```bash
kpm -S make
```

## Usage
Once installed, run 
```bash
make
```
to use

## Uninstallation
To remove make, use:
```bash
kpm -R make
```
